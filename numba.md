## основная информация о библиотеке
Библиотека позволяет оптимизировать код за счет частичной компиляции кода, то есть перевода его в машинный код, что многократно ускоряет его в разы по сравнению с обычной интерпритацией.
Основной действующий элемент это декоратора `@njit()`
его использование сразу компилирует функцию для ее быстрейшего выполнения

он может принимать ряд параметров, таких как:

`nogil = True or False` - может выключить стандартный gil питона и позволить использовать многопоточность

`parallel = True or False` - позволяет включить параллельные вычисления для циклов с использование prange за место стандартного range

`fastmath = True or False` - ускоряет работу с математическими действиями и функциями

`nopython = True or False` - "запрещает"интрепритатору python трогать эту функцию
