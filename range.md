`range()` - принимает от одного до трех int аргументов, возвращает массив с множеством целых чисел от первого, до последнего аргумента, обычно используется для перебора в цикле for
есть 3 варианта использования, от одного до трех аргументов соответственно.


```python
range(**стоп**) берет один аргумент
range(**старт**, **стоп**) берет два аргумента
range(**старт**, **стоп**, **шаг**) берет три аргумента
```
