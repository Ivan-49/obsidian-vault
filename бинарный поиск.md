## основная информация

Используется только к отсортированному списку, 
ход действий:
делит список на 2 части, если тот элемент который он взял удовлетворяет требованиям, то останавливается,
если элемент меньше нужного то уходит в "право"
если больше то в 'лево'

#### пример кода на python

```python
	nums = [1,3,4,5,8,12,14,16,32,43,56]
	request = 32
	def bin_seacrh(lst, req):
	    low = 0
	    high = len(lst)
	    index = None
	    while (low <= high) and (index is None):
	        mid = (low+high)//2
	        if lst[mid] == req:
	            index = mid
	        else:
	            if req < lst[mid]:
	                high = mid-1
	            else:
	                low = mid + 1
	    return index
	
	print(bin_seacrh(nums, request))
```
