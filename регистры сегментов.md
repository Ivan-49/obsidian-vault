Четыре ключевых сегментных регистра: `CS DS ES SS` каждый из которых имеет строгое назначение не может использоваться как регистр общего назначения.

#### CS
CS - Code Segment. Указывает на сегмент кода программы. Регистр [[Управляющие регистры#IP|IP]] Задает смещение внутри него 

#### DS
DS - Data Segment. Для основного сегмента данных(переменные константы). `SI` `DI` или `BX` часто используются для смещения внутри него.

### ES 
ES - Extra Segment. Дополнительный сегмент данных полезен для видео буфера или внешних данных

### SS
SS - Stack Segment. Сегмент стека [[Указатели и индексы#SP|SP]] указывает на вершину [[стек в ассемблере|стека]]
