## Сложение
команда  - `ADD <первый операнд>, <второй операнд>`
Меняет следующие флаги: [[Управляющие регистры#FLAGS|CF,OF,ZF,SF,AF,PF]]
пример: 

```assembler
mov ax, 1 ; ax = 1 bx-мусор
mov bx, 2 ; ax = 1 bx = 2
add ax, bx; ax = 3 bx = 2 
```
## Вычитание
команда - `sub <первый операнд>,<второй операнд>
Меняет следующие флаги: [[Управляющие регистры#FLAGS|CF,OF,ZF,SF,AF,PF]]
пример: 
```
mov ax, 1 ;
mov bx, 1 ; 
sub ax, bx; 

```
