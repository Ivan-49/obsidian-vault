## основная информация
По факту является переборкой массива, и поиск в нем нужного элемента, при чем возвращается не сам элемента а его индекс

#### пример кода

```python
names = ['Ваня','Коля','Вика','Дима','Саша','Олег','Леша','Кирилл','Витя']
search_for = 'Дима'

def line_search(where, what):
	for v in enumirate(where):
		if v[1] == what:
			return v[0]
	return None

print(line_search(names, search_for))
```
