## основная информация 

Этот метод используется для поиска наибольшей подстроки или подмассива обладающего каким-то свойством.



#### подробнее
1. Расширяем подстроку (увеличиваем right, пока в подстроке нет повторяющихся символов.)
2. Сужаем подстроку(увеличиваем left, пока в подстроке есть повторяющиеся символы)

В  данном случае мы ищем подстроку обладающую свойством того что у в ней нет одинаковых символов

Пример кода на python:
```python
def longest_substring(string):
    answer = 0
    left = 0
    right = 0
    char_set = set()
    while (right < len(string)):
        char = string[right]
        if char not in char_set:
            char_set.add(char)
            answer = max(answer, right - left + 1)
            right += 1
        else:
            while char in char_set:
                char_set.remove(string[left])
                left += 1
    return answer
```
