## Основная информация

сегментация  списка на 2 части отсортированную и неотсортированную, циклом мы будем идти по не отсортированной части массива, пока там не останится всего один элемент, при этом мы будем искать наименьший элемент в неотсорт. части списка и заменять его на текущий в левой части, если такой есть, каждый такой уменьшает не сорт. часть
#### пример кода python
```python
list = [5,7,6,9,8,2,4,3,1]
for i in range(len(list)):
    lowest = i
    for j in range(i+1, len(list)):
        if list[j]< list[lowest]:
            lowest = j
      list[i], list[lowest] = list[lowest], list[i]
print(list)
	```
