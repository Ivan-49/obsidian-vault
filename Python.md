## основная информация о библиотеке

Основная библиотека python это набор тех функций который есть в языке изначально их не надо создавать или импортировать, их можно просто вызвать в любой удобный момент

### ссылки на более подробные части 

`len()` - длинна списка [[len]]
`range()` - множество [[range]]
`.split('')` - разделение строк [[split]] 
`enumerate(list)` - индекс и значение элемента списка [[enumerate]]
`''.join(list)`  -  соединение списка из строк в одну строку [[join]]
