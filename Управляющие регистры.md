
## IP
IP - Instruction Pointer. Задает смещение в сегменте кода.


## FLAGS
Флаги состояния.
Является 16битным регистром. В 8086 значимые биты:
* CF(0) - Carry Flag, Флаг переноса
* PF(2) - Paruty Flag
* AF(4) - Auxiliary carry Flag
* ZF(6) - Zero Flag
* SF(7) - Sign Flag
* TF(8) - Trap Flag
* IF (9) - Interrupt enable
* BF (10) - Direction
* OF(11) - Overflow Flag
